# NLP Exercise

For this exercise, we will ask you to complete a small NLP data science project using a template notebook and publicly-available data. The example project will use the Blog Authorship Corpus, originally from Kaggle, modified and provided in this repository. The notebook `template.ipynb` will seve as a template for you to complete the four phases of the project:

1. Data Loading
2. Deata Exploration
3. Modeling
4. Evaluation
